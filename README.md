# 🔴 Classic Checkers (User vs User)

> A modern, minimal checkers application built using **React + Vite**, featuring **Player vs Player** gameplay. The UI follows a warm wooden board aesthetic with cream and dark brown colors.

---

## Features

- **Player vs Player** (Local Multiplayer)
- Checkers logic built from scratch (including forced jumps, King promotion, multi-jumps)
- Move history and turn tracking
- Clean, minimal UI inspired by a wooden board
